本目录包含从 Claude Code 开发记录中整理的面试题和相关材料，针对一个使用 Turborepo + Next.js 14 + Prisma + tRPC + AI Agent 技术的 SaaS 平台项目。

### 1. `interview_questions_organized.md` (新整理)
- **内容**: 结构化整理的 24 道面试题，按技术领域分类
- **特点**:
  - 按 Monorepo、多租户、实时协作、AI Agent、系统设计等专题组织
  - 每道题包含考察点和回答要点
  - 提供面试准备建议和技术深度评估

### 2. `interview_questions.md` (原始文件)
- **内容**: 原始的 25 道定制化面试题
- **特点**:
  - 包含基础确认题、深度追问题、扩展场景题
  - 每题有考察点、评分要点、你的优势三个部分
  - 覆盖 5 个专题：Monorepo、多租户、实时协作、AI Agent、系统设计

### 3. `project_knowledge_graph.md`
- **内容**: 项目知识图谱
- **特点**:
  - 技术栈与架构模式分析
  - 6 个关键架构决策清单（含置信度评估）
  - 交叉印证发现（Gap、孤岛、方案变更）
  - 技术域分布和面试价值评估

### 4. `story_cards.md`
- **内容**: 5 张 STAR 故事卡
- **特点**:
  - 基于高价值决策的完整 STAR 叙述
  - 每张卡片 200-300 字，第一人称
  - 包含情境、任务、行动、结果四个部分
  - 提供故事叙述技巧指导

## 技术栈覆盖

### 核心架构
- **Monorepo**: Turborepo（vs Lerna/Nx）
- **前端**: Next.js 14 App Router
- **后端**: tRPC + TypeScript
- **数据库**: PostgreSQL + Prisma
- **缓存**: Redis

### 高级特性
- **多租户**: PostgreSQL schema 隔离
- **实时协作**: Yjs (CRDT) + WebSocket
- **AI Agent**: 工具模式 + 执行引擎 + 记忆管理
- **部署**: Vercel + Railway + 托管数据库

## 面试价值

### 高价值展示点
1. **现代技术栈组合**: 展示对前沿技术的掌握
2. **复杂场景处理**: 多租户、实时协作、AI Agent 集成
3. **架构演进思维**: 考虑扩展性、迁移路径
4. **工程化实践**: 类型安全、测试策略、CI/CD

### 适合职级
- 资深全栈工程师
- 架构师
- 技术负责人

## 使用建议

### 面试准备
1. **重点准备**: 选择 3-5 个最有信心的专题深入准备
2. **故事练习**: 基于 STAR 故事卡练习技术决策叙述
3. **技术深度**: 不仅要会使用，还要理解原理和限制

### 风险规避
1. **安全话题**: 承认当前实现基础，展示安全意识和改进计划
2. **监控运维**: 说明生产就绪性的差距，强调监控设计思路
3. **性能优化**: 展示优化意识，即使具体实现不完整
